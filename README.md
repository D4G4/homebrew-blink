# Homebrew Tap for Blink

Smart 20-20-20 eye break reminder for macOS.

## Install

```bash
brew tap D4G4/blink
brew install --cask blink
```

## Update

```bash
brew upgrade --cask blink
```

## Uninstall

```bash
brew uninstall --cask blink
```

## About

[Blink](https://github.com/D4G4/blink) detects your work patterns and prompts eye breaks at the right moment — never mid-thought.
